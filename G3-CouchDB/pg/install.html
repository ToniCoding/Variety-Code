Crear base de datos
Comando: curl -u admin:1234 -X PUT http://127.0.0.1:5984/supermercado
Borrar base de datos
Comando: curl -u admin:1234 -X DELETE http://127.0.0.1:5984/supermercado
Insertar documento
Comando: curl -u admin:1234 -X PUT http://127.0.0.1:5984/supermercado/UUID -d '{"nombre": "Manzana", "tipo" : "fruta", "precio" : 5}'
Actualizar documento
Comando: curl -u admin:1234 -X PUT http://127.0.0.1:5984/supermercado/98c003b03bc8aa87cb05983d1c000713 -d '{"_rev": "1-eba25568090eb2dfffad770b55147a67","nombre": "Manzana", "tipo" : "fruta", "precio" : 4}'
Borrar documento
Comando: curl -u admin:1234 -X DELETE http://127.0.0.1:5984/supermercado/98c003b03bc8aa87cb05983d1c000713?rev=2-298fdb46385be60609b242b3e5cc3566
